# Unocss Intellij

> <a href="https://github.com/unocss/unocss">Unocss</a> Support for Intellij Based IDEs

## Features

- Auto complete
- Syntax highlight
- Documentation
- Directive support (e.g. `@apply, @screen, @theme`)
- Attributify support
- Line marker for colors and icons
